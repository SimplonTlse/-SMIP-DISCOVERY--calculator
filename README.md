# -SMIP-DISCOVERY--calculator

# Objectifs
Réaliser une calculatrice. L'utilisateur pourra soit cliquer sur des boutons soit taper directement au clavier ses instructions. Les opérations à mettre sont : 
* Addition
* Soustraction
* Multiplication
* Division
* Réinitialisation (CE)
